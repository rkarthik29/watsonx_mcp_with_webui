## Claude-Watsonx-Studio Quick Start

1. Edit `.env` file and add your API keys:
   - ANTHROPIC_API_KEY
   - WATSONX_APIKEY
   - WATSONX_PROJECT_ID
   - WATSONX_INSTANCE_CRN
   - WATSONX_URL

2. Run:
   docker-compose up -d

3. Open http://localhost:3000
